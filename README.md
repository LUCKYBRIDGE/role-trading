# 역할 선택 생활교육 카드 트레이너

초보 교사가 역할 선택 생활교육의 핵심 이론을 카드로 공부하고, 다양한 교실 문제상황에서 `관찰자`, `사건 정리자`, `경계 설정자`, `시스템 연결자` 중 어떤 자리에 설지 선택형 퀴즈로 연습하는 정적 웹앱입니다.

이 웹은 리피스평화교육연구소 대표이자 유튜브 `학생생활교육 마스터클래스` 운영자인 정진 소장님의 교육과 자료를 바탕으로 제작되었습니다.

- 유튜브 채널: https://www.youtube.com/@Social_Profiler/featured

핵심 관점은 교사가 아이를 통제하려 하며 1대1 대치로 들어가는 대신, 교실 공동체의 기준을 제시하고 학생이 공동체 안에서 자기 역할을 다시 설정할 수 있도록 돕는 것입니다.

역할은 태도가 아니라 위치입니다. 교사는 응보적으로 혼내는 사람이 아니라 공동체를 대표하여 기준을 말하는 사람으로 개입합니다.

## 실행

별도 빌드 없이 브라우저에서 `index.html`을 열면 됩니다.

## 구성

- `이론 카드`: 역할 선택 생활교육의 기본 개념을 플래시카드처럼 앞뒤로 넘겨 익힙니다.
- `상황 퀴즈`: 교실 장면을 보고 먼저 설 역할을 선택한 뒤, 추천 행동과 언어를 확인합니다.
- `문장 카드`: 역할별 기준 문장을 한 장씩 넘기며 소리 내어 연습합니다.

## 자료 구조

문제상황 데이터는 `app.js`의 `scenarios` 배열에 들어 있고, 역할별 문장 데이터는 `sentences` 배열에 들어 있습니다. 추후 JSON 파일이나 서버 데이터로 분리하기 쉽도록 역할, 단계, 행동, 문장을 나누어 두었습니다.
